# FOMO Sentiment Analysis Pipeline
مشروع هندسة بيانات (Data Engineering) متكامل.

## الوصف:
نظام أوتوماتيكي (ETL Pipeline) يقوم بـ:
1. استخراج (Extract) البيانات الخام من Kaggle.
2. تنظيف ومعالجة (Transform) البيانات باستخدام Python.
3. تحميل (Load) البيانات المحدثة لتظهر فوراً على لوحة تحكم Power BI.

## الأدوات المستخدمة:
- Python (Pandas)
- Power BI
- Kaggle API

## التشغيل:
- قم بتشغيل الكود `pipeline.py` لتحديث البيانات المربوطة بالداشبورد.
